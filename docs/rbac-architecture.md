# RBAC Architecture

## Roles

### Developers
- Execute scans
- View findings
- Fix vulnerabilities

### Reviewers
- Review issues
- Validate remediation
- Comment on findings

### Security Admins
- Configure security policies
- Manage users/groups
- Configure quality gates
- Manage rule profiles

## Security Controls
- Authentication required
- Least privilege access
- Token-based CI/CD authentication
- Project-level permissions
